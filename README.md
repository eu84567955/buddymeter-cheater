# Buddymeter Cheater
讓你能手動定義Buddy meter的分數，並能自動答對所有題目。 

## Demo
![](https://i.imgur.com/3XAyMmk.gif)

## Features
- 自動答完並答對所有題目（獲得滿分十分）。
- 自訂最終分數（0~10，可為小數），並自動答完所有題目。
